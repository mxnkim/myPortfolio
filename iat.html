An interactive tactile interface that invites participants to touch the luminous fiber-optic grass. Different light patterns are created depending on how participants touch Grass. This explores natural touching or stroking gestures experimenting soft-circuit techniques with alternative conductive materials: conductive paints and fabrics.

The design of this project is to explore an interactive origami light that transforms its shape in the users hand and creates light patterns depending on the form. It aims to create a playful, poetic interface that evokes user creativity through interaction. Tessella is arranged as a matrix of light and sensors on a flexible tessellated structure.

Fillin (Hugging Tree) is a life-size cherry blossom tree, which when touched (or hugged) on the tree trunk, cherry blossoms will be illuminated vibrantly. Fillin (Hugging Tree) is meant to have a simplistic and natural looking physical model. The main focus of the installation is designated to be the user experience.

This is an interactive immersive installation that consists of 50 resin raindrops, hand casted. Illuminating Raindrops create light patterns and movement patterns depending on participants movement around them. Rain Drops creates relaxed/immersive experience like being under the rain as well as playful experience touching rain drops.
